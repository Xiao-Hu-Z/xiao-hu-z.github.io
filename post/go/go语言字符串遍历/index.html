Go语言——字符串遍历

Posted by xiaohu on Thursday, November 11, 2021

字符串遍历

在go语言中支持两个字符类型，一个是byte(实际上是uint8的别名)，代表UTF-8字符串的单个字节的值，另一个是rune（为int32），代表单个Unicode字符。

utf-8遍历

package main

import "fmt"

func main() {
    str := "Hello,世界"
    fmt.Println("len:", len(str))
    for i := 0; i < len(str); i++ {
        ch := str[i]
        fmt.Println(ch)
    }
}

每个中文字符在UTF-8中占3个字节，而不是1个字节

转化为字面值输出：

package main

import (
	"fmt"
)

func main() {
	str := "Hello,世界"
	fmt.Println("utf-8遍历  格式化打印")
	for _, ch1 := range str {
		fmt.Printf("%q", ch1) //单引号围绕的字符字面值，由go语法安全的转义
	}
}
//utf-8遍历  格式化打印
//'H''e''l''l''o'',''世''界'%

unicode遍历

以Unicode字符方式遍历时，每个字符的类型是rune，而不是byte。rune类型在go语言中占用四个字节

package main

import "fmt"

func main() {
	str := "Hello,世界"
	for _, ch1 := range str {
		fmt.Println(ch1)
	}
}

输出：

72
101
108
108
111
44
19990
30028

转化为字面值输出：

package main

import (
	"fmt"
)

func main() {
	str := "Hello,世界"
	fmt.Println("unicode遍历  转化输出格式")
	for _, ch2 := range str {
		fmt.Printf(string(ch2))
	}
}

//unicode遍历  转化输出格式
//Hello,世界%
